Feedstock for [acts](https://github.com/acts-project/acts) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Experiment-independent toolkit for particle track reconstruction

## Install

```bash
conda install -c hep-forge -c conda-forge acts
```

Feedstock for [acts](https://acts.readthedocs.io/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Experiment-independent toolkit for particle track reconstruction

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `46.8.1` |
| linux-arm64 (`linux-aarch64`) | ✅ `46.8.1` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge acts
```
